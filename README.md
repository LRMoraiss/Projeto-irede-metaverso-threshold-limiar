# Projeto-irede-metaverso-threshold-limiar
 Survival extraction com IA adaptativa — Mundo 1: A Floresta. 

PROJETO: Meu Primeiro Ambiente VR
Residência em TIC 29 - Web 3.0
Aluno: Luciano Rodrigues de Morais

DESCRIÇÃO
Cena VR de naufrágio. O jogador acorda sozinho numa praia desconhecida.
As últimas memórias são de um cruzeiro. O bote furado e os pertences 
espalhados sugerem que algo deu muito errado. A floresta à frente é a 
única saída.

OBJETOS DA CENA
1. Areia - Plano de chão (terreno da praia)
2. Bote_Furado - Cubo achatado (bote inflável furado)
3. Remo_Quebrado - Cilindro fino em ângulo (remo caído na areia)
4. Mala_Aberta - Esfera achatada (mala amassada)
5. Colete_Salva_Vidas - Cilindro achatado (colete de emergência)
6. Sinalizador_Flare - Cilindro fino vertical (sinalizador fincado na areia)

STACK TÉCNICA
- Unity 6 (6000.3.13f1)
- Universal Render Pipeline (URP)
- Meta XR SDK v201
- Meta Building Blocks - Camera Rig

COMO EXECUTAR
1. Abra o projeto no Unity 6
2. Abra a cena Assets/Scenes/Praia-begin
3. Pressione Play no Unity Editor
4. A movimentação funciona via teclado/mouse no PC
